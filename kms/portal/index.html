🍄 AAStar 社区节点

买一块 NXP FRDM-IMX93 主板,自助起你自己的 KMS / DVT 节点,加入 Mycelium 门限网络。不需要懂 CLI / 编译 / Git。

⚠️ 脚手架版 —— 板子出厂即 NXP 定制 Linux(含 OP-TEE),无需重刷系统;只跑一条 installer 装我们的组件。gasless 一键注册待接线。

三步走

① 拿到板子
买 NXP FRDM-IMX93(出厂自带 OP-TEE Linux),或找 AAStar 要预刷板(邮寄)。
② 装上节点
联网 → 跑一条 installer(下面选一种角色)→ 自动装 KMS/DVT。不刷 Linux。
③ 浏览器配置
打开板子 IP:8088 → 填几项 → 首启在 TEE 内生成密钥 → 加入门限网络。

① 选一种节点角色

同一套组件,4 种角色只是「装哪些服务 + 签名方式」的差异。选好后下面的命令会自动带上 PROFILE。

② 一条命令装上

板子联网后,SSH 上去跑(NXP 出厂 Linux 已含 OP-TEE,无需重刷):

curl -fsSL https://raw.githubusercontent.com/AAStarCommunity/AirAccount/main/kms/deploy/aastar-node-installer.sh | sudo PROFILE=combined-independent bash

装完:有 KMS 的角色会在 :8088 起配置向导;DVT 在 :8080、KMS 在 :3000。想先看它会做什么,加 DRY_RUN=1 只打印计划不改动。

预刷板(最省心)★★★ 零操作

不想碰命令行?找 AAStar 要预刷板(已装好邮寄)。通电 → 联网 → 浏览器打开 http://<板子IP>:8088 跑向导即可。

整盘镜像自刷⏳ 待 CI

.wic 整盘镜像(balenaEtcher 拖拽刷)待 Yocto CI 就绪。现阶段用上面的 installer 即可 —— 出厂 Linux 人人相同,不需要重刷整盘。

配置向导会问什么

项说明
Setup Token板子串口/日志打印的一次性令牌(证明你能访问板子)
你的域名(rpId)⚠️ 必须是你自己的域名(passkey 身份独立),不能用 aastar.io
Operator 地址链上注册用;没 ETH 也行,AAStar 可 gasless 代付
网络测试网 Sepolia(先试)/ 主网

向导做:私钥全程首启在板生成,不显示、你本地只填 config(无私钥)。 保护级别按角色:KMS 密钥 与 combined-tee 的 DVT = TEE 密封,永不出 TEE; dvt-only / combined-independent 的 DVT = 本地 EIP-2335 加密 keystore(在板生成、密码走 tmpfs,非 TEE 密封)。见 初始化配置设计。
