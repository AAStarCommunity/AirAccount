🍄 AAStar 社区节点

买一块 MX93 主板,自助起你自己的 KMS + DVT 节点,加入 Mycelium 门限网络。不需要懂 CLI / 编译 / Git。

⚠️ 脚手架版(Phase 3 起步)—— 下载链接与 gasless 一键注册待接线;当前引导到 GitHub Releases + 自助向导。

三步走

① 拿到板子
买 NXP FRDM-IMX93,或找 AAStar 要预刷板(邮寄)。
② 装上节点
预刷板直接开机;或刷镜像 / 跑一条 installer 命令。
③ 浏览器配置
开机联网 → 打开板子 IP:8088 → 填几项 → 完成,加入门限网络。

选一档傻瓜度

A · 预刷板(最省心)★★★ 零刷机

AAStar 刷好板子邮寄给你。通电 → 联网 → 浏览器打开 http://<板子IP>:8088 跑配置向导。

适合:不想碰刷机的社区。找 AAStar 申请预刷板。

B · 下载镜像自刷 ★★ 有 GUI

下 .wic 整盘镜像 → balenaEtcher 拖拽刷 SD/eMMC → 开机 → 同 A 的向导。

⏳ 镜像发布待 CI(Yocto)就绪。暂用下面 C 档 installer。

C · 基础镜像 + 一条命令 ★ 需一条命令

刷 NXP 官方 FRDM-IMX93 OP-TEE 基础镜像(含 OP-TEE),然后: 
curl -fsSL https://raw.githubusercontent.com/AAStarCommunity/AirAccount/main/kms/deploy/aastar-node-installer.sh | sudo bash
自动装 KMS + DVT + 配置向导。详见 Phase 2 镜像文档。
